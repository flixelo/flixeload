# FlixeloAd SDK Example

A complete example app demonstrating all features of the FlixeloAd SDK.

## Features Demonstrated

- ✅ Banner ads (Standard 320x50)
- ✅ Medium rectangle ads (300x250)
- ✅ Native ads with custom styling
- ✅ Interstitial ads
- ✅ Rewarded video ads
- ✅ Ad targeting with keywords
- ✅ Event callbacks

## Running the Example

```bash
cd example
flutter pub get
flutter run
```

## Configuration

The example uses test mode with demo credentials:
- App ID: `com.example.demo`
- API Key: `demo_api_key`
- Test Mode: `true`

For production use, replace with your actual credentials from [flixeload.com](https://flixeload.com).
